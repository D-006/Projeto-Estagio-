**2.Requisitos**

Poder criar conta, poder fazer login, poder comparar componentes, analisar e comparar preços das peças de PC, obter recomendações das melhores peças de acordo com a build que o utilizador, poder salvar as builds.



**2.1.Requisitos Não Funcionais**

**O sistema deverá ser rápido, seguro e fácil de utilizar, funcionando corretamente nos principais navegadores para computador. O site deverá garantir a integridade dos dados, permitir futuras expansões e possuir uma estrutura organizada para facilitar manutenção e atualizações. Além disso, o sistema deverá suportar execução através de Docker e armazenar permanentemente os dados na base de dados.**
